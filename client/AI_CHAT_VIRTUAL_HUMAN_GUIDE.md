# AI对话页面虚拟人集成指南

## 🎯 功能概述

您的AI对话页面现在已经集成了虚拟人SDK功能！您可以在原有的AI对话界面中看到真实的虚拟人，并且虚拟人会根据AI的回复内容进行相应的语音播报和动作。

## ✨ 新增功能

### 1. 虚拟人切换
- **动画虚拟人**：原有的3D动画虚拟人（默认）
- **真实虚拟人**：基于讯飞虚拟人SDK的真实虚拟人

### 2. 智能交互
- 虚拟人会根据AI回复的内容自动进行语音播报
- 支持表情和动作的智能识别
- 完整的语音交互功能

### 3. 大模型集成
- 已配置您的大模型对话功能
- 集成了知识库功能
- 使用deepseek-r1模型

## 🚀 使用方法

### 1. 访问AI对话页面
```
http://localhost:3000/chat
```

### 2. 切换虚拟人模式
1. 在页面右上角找到虚拟人区域
2. 点击"动画虚拟人"按钮切换到"真实虚拟人"
3. 等待虚拟人连接成功（显示"已连接"状态）

### 3. 开始对话
1. 在输入框中输入问题
2. 点击发送或按Enter键
3. 观察虚拟人的反应：
   - 动画虚拟人：显示表情变化
   - 真实虚拟人：进行语音播报和动作

## 🎭 虚拟人行为

### 表情识别
虚拟人会根据AI回复内容自动调整表情：
- **欢迎/感谢** → 开心表情
- **思考/分析** → 思考表情
- **抱歉/错误** → 困惑表情
- **惊喜/成功** → 兴奋表情
- **专业内容** → 专业表情

### 语音播报
- 动画虚拟人：使用浏览器语音合成
- 真实虚拟人：使用讯飞TTS语音合成
- 支持语音开关控制

## 🔍 状态指示

### 连接状态
- **连接中...**：正在建立连接
- **已连接**：虚拟人已就绪
- **未连接**：连接失败或未启动

### 交互状态
- **说话中**：虚拟人正在播报
- **聆听中**：正在接收用户输入
- **思考中**：AI正在处理请求

## 🛠️ 故障排除

### 1. 虚拟人无法连接
**可能原因：**
- 网络连接问题
- API配置错误
- 虚拟人ID无效

**解决方案：**
- 检查网络连接
- 确认API配置正确
- 联系技术支持验证ID

### 2. 虚拟人不说话
**可能原因：**
- 语音功能被禁用
- 浏览器自动播放限制
- TTS配置问题

**解决方案：**
- 检查语音开关状态
- 点击页面任意位置解除限制
- 确认TTS配置正确

### 3. 大模型不工作
**可能原因：**
- 大模型未开启
- 知识库配置错误
- 提示词格式问题

**解决方案：**
- 确认大模型已开启
- 检查知识库配置
- 验证提示词格式
