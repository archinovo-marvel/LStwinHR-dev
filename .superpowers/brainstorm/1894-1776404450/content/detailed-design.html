Flex视觉平衡 - 详细设计

三个核心功能卡片布局调整方案

设计目标

  • 卡片适当拉长，弥补左侧空白
  • 视觉平衡，非严格三等分
  • 保持现有卡片样式和交互效果
  • 响应式适配：桌面端水平排列，移动端垂直堆叠

代码改动位置

1. HomePage.js - 第562-586行（当前结构）
// 当前代码（使用Ant Design栅格）
<Row gutter={[20, 20]}>
  {features.map((feature, index) => (
    <Col xs={24} sm={12} lg={6} key={index}>
      <motion.div>
        <FeatureCard>...</FeatureCard>
      </motion.div>
    </Col>
  ))}
</Row>

问题：每卡片6列，三卡片共18列，左侧留6列空白

2. 新设计（Flex容器替换）
// 移除Row/Col，添加CSS类名
<div className="features-flex-container">
  {features.map((feature, index) => (
    <motion.div
      className="feature-flex-item"
      initial={{ opacity: 0, y: 20 }}
      animate={{ opacity: 1, y: 0 }}
      transition={{ duration: 0.3, delay: index * 0.1 }}
      key={index}
    >
      <FeatureCard ... />
    </motion.div>
  ))}
</div>
3. CSS样式（添加到FeaturesSection styled component后）
