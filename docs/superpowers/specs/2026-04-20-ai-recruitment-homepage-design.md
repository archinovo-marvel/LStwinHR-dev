# AI招聘面试网站首页设计规范

## 概念与愿景

高端品牌campaign风格首页，灵感来自ESTUDIO ANONIMO的编辑设计语言。大量留白、杂志感排版、克制动效。整体传达专业、克制、高端的品牌调性，如同翻阅一本精装的招聘行业杂志。

## 设计语言

### 美学方向
- **参考**: ESTUDIO ANONIMO编辑风格 + 高端品牌campaign
- **核心**: 超大衬线字标题、极简导航、重构留白、杂志感非对称布局
- **禁止**: 模板味、普通SaaS hero、居中布局、AI紫蓝渐变

### 色彩系统
| 用途 | 色值 | 名称 |
|------|------|------|
| 背景 | `#FAF8F5` | 米白 |
| 次级背景 | `#F0EDE8` | 暖灰白 |
| 主文字 | `#1A1A1A` | 近黑 |
| 次级文字 | `#6B6B6B` | 中灰 |
| 边框/分割 | `#E0DDD8` | 浅灰 |
| 强调 | `#2C2C2C` | 炭灰 |
| 点缀 | `#8B7355` | 棕褐 |

### 字体
- **标题**: Noto Serif SC (中文衬线), Playfair Display (英文衬线)
- **正文**: Noto Sans SC (中文无衬线), Inter (英文无衬线)
- **字号策略**: 超大标题 80-120px，section标题 48px，正文 16px

### 空间系统
- **留白**: 大面积留白，section间距 120-200px
- **边距**: 内容区 max-width 1400px，左右 padding 80px (desktop)
- **非对称**: 文字与视觉元素刻意错位，打破网格

### 动效哲学
- **克制**: 仅在关键交互处使用微动效
- **入场**: 单次 fade-in + translateY, 800ms ease-out
- **悬停**: 轻微色变、underline展开，不做scale
- **禁止**: 弹跳、脉冲、循环动画

## 组件清单

### Navigation
- 高度 80px，固定顶部
- logo左侧，菜单项目右侧
- 菜单: 关于我们 / 核心服务 / 联系我们
- 底部1px边框分隔
- 背景: 透明或半透明

### Hero Video
- `<video>` 标签，autoplay muted loop playsInline
- `object-fit: cover` 全屏覆盖
- 暗色遮罩 `rgba(0,0,0,0.3)` 叠加
- 文字 z-index 10

### Section Title
- 超大衬线字体
- `font-size: clamp(48px, 8vw, 96px)`
- `font-weight: 400` (非粗体)
- 中文字体 Letter-spacing 正常

### Feature Card
- 无边框、无阴影
- 悬停: 文字颜色变化 + 左侧竖线出现
- 图片: 灰度处理，悬停时恢复彩色

### Stats Block
- 超大数字 + 小字标签
- 数字使用等宽字体
- 非居中，左对齐

### CTA Button
- 文字链样式，下划线展开动效
- 禁止: 填充按钮、渐变背景、圆角

## 技术实现

- **框架**: React 18 + styled-components
- **动效**: framer-motion (克制使用)
- **字体**: Google Fonts CDN (Noto Serif SC, Noto Sans SC)
- **视频**: 本地MP4或远程URL
- **图片**: Picsum高质量占位图
