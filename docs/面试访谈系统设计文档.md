## 一、系统概述

### 1.1 系统定位

AI面试访谈系统是企业招聘流程中的智能化面试评估工具，通过AI驱动的结构化面试，对候选人进行IQ（智商）、EQ（情商）、AQ（逆商）、MQ（德商）四维度的综合评估。

### 1.2 核心功能

| 功能模块 | 功能描述 |
|----------|----------|
| 智能提问 | 基于候选人简历和目标岗位，AI动态生成个性化面试问题 |
| 实时评分 | 每道题目回答后，AI即时进行四维度评分并生成评语 |
| 追问机制 | IQ维度支持递进式追问，验证专业知识的深度和经验真实性 |
| 综合报告 | 面试结束后自动生成包含四维度分析的综合评估报告 |
| 应届生适配 | 自动识别应届生/无经验候选人，调整提问策略 |

### 1.3 技术特点

- **AI驱动**：问题生成、评分、报告撰写全程由本地LLM完成
- **个性化**：根据简历内容、岗位要求、候选人背景动态调整问题
- **结构化**：固定12道题目，覆盖IQ/EQ/AQ/MQ四个维度
- **实时性**：每题回答后即时评分，面试结束即可查看报告

### 3.3 问题生成机制

系统通过构建结构化Prompt，调用本地LLM动态生成面试问题。Prompt包含以下核心要素：

| Prompt组成要素 | 说明 |
|----------------|------|
| 维度核心考察点 | 如"专业知识深度、经验真实度、岗位胜任力" |
| 本题考察方向 | 如"专业知识深度考察" |
| 评分标准参考 | 8-10分、4-7分、0-3分的评价标准 |
| 候选人简历摘要 | 从简历中提取的关键信息 |
| 目标岗位信息 | 岗位名称、描述、技能要求 |
| 候选人背景类型 | 应届生/有工作经验 |
| 前序问答上下文 | IQ维度的追问需要前序问答 |
| 强制要求 | 问题必须针对考察方向，只输出一个问题 |

## 四、四维度评分体系

### 4.1 评分体系概览

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        面试评分体系（满分 100 分）                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ┌─────────────┐   ┌─────────────┐   ┌─────────────┐   ┌─────────────┐   │
│   │   IQ 智商   │   │   EQ 情商   │   │   AQ 逆商   │   │   MQ 德商   │   │
│   │   50 分     │   │   20 分     │   │   15 分     │   │   15 分     │   │
│   │   (50%)     │   │   (20%)     │   │   (15%)     │   │   (15%)     │   │
│   └─────────────┘   └─────────────┘   └─────────────┘   └─────────────┘   │
│                                                                             │
│   每维度3题，每题满分10分                                                    │
│   维度得分 = (三题平均分 / 10) × 维度权重                                    │
│                                                                             │
│   计算示例：                                                                 │
│   IQ三题得分：8, 9, 7 → 平均8分 → IQ得分 = (8/10) × 50 = 40分              │
│   EQ三题得分：7, 8, 6 → 平均7分 → EQ得分 = (7/10) × 20 = 14分              │
│   AQ三题得分：8, 7, 8 → 平均7.67分 → AQ得分 = (7.67/10) × 15 = 11.5分      │
│   MQ三题得分：9, 8, 9 → 平均8.67分 → MQ得分 = (8.67/10) × 15 = 13分        │
│   总分 = 40 + 14 + 11.5 + 13 = 78.5分                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 4.2 四维度定义对比

| 维度 | 英文全称 | 中文名称 | 满分 | 权重 | 核心考察点 |
|------|----------|----------|------|------|------------|
| IQ | Intelligence Quotient | 智商 | 50分 | 50% | 专业知识深度、经验真实度、岗位胜任力 |
| EQ | Emotional Quotient | 情商 | 20分 | 20% | 情绪管理、同理心、人际沟通 |
| AQ | Adversity Quotient | 逆商 | 15分 | 15% | 抗压性、复原力、从失败中学习 |
| MQ | Moral Quotient | 德商 | 15分 | 15% | 职业操守、原则性、诚信与社会责任 |

### 4.3 实时评分四维度

每道题目回答后，AI从以下四个维度进行评分（各1-10分）：

| 评分维度 | 英文 | 说明 | 高分特征 | 低分特征 |
|----------|------|------|----------|----------|
| 相关性 | relevance | 是否切题 | 紧扣问题核心 | 答非所问 |
| 深度 | depth | 内容深度 | 具体细节、数据、反思 | 笼统空泛 |
| 清晰度 | clarity | 表达清晰度 | 逻辑清晰、表达流畅 | 逻辑混乱 |
| 专业性 | professionalism | 专业素养 | 成熟心态、专业表现 | 不成熟、缺乏素养 |

## 五、IQ维度详解

### 5.1 维度定义

| 属性 | 值 |
|------|-----|
| 维度代码 | iq |
| 中文名称 | 智商IQ |
| 满分 | 50分 |
| 权重占比 | 50% |
| 核心考察点 | 专业知识深度、经验真实度、岗位胜任力 |

### 5.2 三道题目设计

| 题号 | 考察方向 | 提问策略 | 追问关系 |
|------|----------|----------|----------|
| Q1 | 专业知识深度考察 | 从简历中选择核心技能或项目，要求详细阐述技术细节、实现方案、难点攻克 | 无前序，独立提问 |
| Q2 | 经验真实度验证 | 必须针对Q1回答追问，追问具体细节、数据支撑、个人贡献 | 追问Q1的回答 |
| Q3 | 岗位胜任力评估 | 将候选人能力与目标岗位要求关联，评估核心能力匹配度 | 关联岗位JD |

### 5.3 评分标准

**Q1 - 专业知识深度考察**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 专业知识扎实，能深入阐述技术细节，回答与简历经历高度吻合 |
| 4-7分 | 有基本专业知识，能说明大概，但细节不够深入或与简历有出入 |
| 0-3分 | 专业知识薄弱，回答含糊，与简历描述明显不符 |

**Q2 - 经验真实度验证**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 能提供具体项目细节、数据和成果，经验真实可信，逻辑自洽 |
| 4-7分 | 能描述基本经历，但细节不够具体，部分内容缺乏佐证 |
| 0-3分 | 描述笼统空泛，无法提供具体细节，经验真实性存疑 |

**Q3 - 岗位胜任力评估**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 展现岗位所需核心能力，有清晰职业规划，与岗位高度匹配 |
| 4-7分 | 具备部分岗位能力，匹配度一般，需要进一步培养 |
| 0-3分 | 核心能力不足，与岗位要求差距较大 |

### 5.4 关键词评分规则

**正向关键词（每个+2分）**：
负责、主导、优化、实现、设计、架构、部署、测试、数据、成果、提升、降低、解决、方案、具体、项目、技术、细节、量化、指标

**负向关键词（每个-2分）**：
大概、可能、好像、不太记得、差不多、应该、感觉、别人做的、不清楚

### 5.5 应届生适配策略

| 题号 | 有工作经验者 | 应届生/无经验者 |
|------|--------------|-----------------|
| Q1 | 从实际工作项目中考察专业深度和经验真实度 | 从专业技能、课程项目、竞赛经历中考察知识深度 |
| Q2 | 追问工作细节验证经验真实性 | 追问项目细节验证参与度 |
| Q3 | 将候选人能力与岗位要求进行匹配分析 | 将候选人能力与岗位要求进行匹配分析 |

## 六、EQ维度详解

### 6.1 维度定义

| 属性 | 值 |
|------|-----|
| 维度代码 | eq |
| 中文名称 | 情商EQ |
| 满分 | 20分 |
| 权重占比 | 20% |
| 核心考察点 | 情绪管理、同理心、人际沟通 |

### 6.2 三道题目设计

| 题号 | 考察方向 | 有经验者情境 | 应届生情境 |
|------|----------|--------------|------------|
| Q4 | 冲突处理与情绪认知 | 跨部门协作冲突、客户沟通分歧 | 团队合作冲突、社团活动分歧、宿舍相处 |
| Q5 | 艰难沟通与同理心 | 给下属/客户传递坏消息 | 给同学/老师传递困难消息 |
| Q6 | 自我情绪觉察与调节 | 工作压力下的情绪管理 | 学业压力、考试焦虑的情绪调节 |

### 6.3 评分标准

**Q4 - 冲突处理与情绪认知**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 清晰剖析情绪变化，主动管理情绪、换位思考并达成双赢，有深刻反思 |
| 4-7分 | 能识别双方情绪，描述了通过沟通寻求解决的具体步骤，结果尚可 |
| 0-3分 | 只描述事件，未提及自身或他人情绪，解决方案简单或回避 |

**Q5 - 艰难沟通与同理心**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 有完整策略：预判情绪、选择场合、阐述背景、开放沟通、提供支持 |
| 4-7分 | 会考虑时机、方式，计划解释原因并愿意倾听反馈 |
| 0-3分 | 仅考虑传递信息本身，未顾及团队接收感受 |

**Q6 - 自我情绪觉察与调节**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 有清晰自我预警指标，并有系统、主动的调整方案（如正念、任务重构、求助） |
| 4-7分 | 能列举具体身心信号，有常规放松方式（如运动、倾诉） |
| 0-3分 | 难以识别或描述模糊，调整措施单一无效 |

### 6.4 关键词评分规则

**正向关键词（每个+2分）**：
换位思考、共情、理解、倾听、沟通、感受、情绪、反思、包容、体谅

**负向关键词（每个-2分）**：
不管、无所谓、不在乎、懒得、随便

## 七、AQ维度详解

### 7.1 维度定义

| 属性 | 值 |
|------|-----|
| 维度代码 | aq |
| 中文名称 | 逆商AQ |
| 满分 | 15分 |
| 权重占比 | 15% |
| 核心考察点 | 抗压性、复原力、从失败中学习 |

### 7.2 三道题目设计

| 题号 | 考察方向 | 有经验者情境 | 应届生情境 |
|------|----------|--------------|------------|
| Q7 | 面对失败与反思 | 项目失败、职场挫折 | 学业挫折、竞赛失败、求职受挫 |
| Q8 | 逆境中的应变与决策 | 项目计划突变、资源削减 | 考试计划被打乱、活动取消 |
| Q9 | 持续高压下的动力维持 | 长期项目压力、多任务并行 | 考研/求职季的持续压力 |

### 7.3 评分标准

**Q7 - 面对失败与反思**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 坦诚剖析，重点阐述从中学到的具体东西及如何应用后续改进，展现成长型思维 |
| 4-7分 | 客观描述，承担自身责任，能总结出基本经验教训 |
| 0-3分 | 回避或淡化失败，归咎外因，未体现个人反思与学习 |

**Q8 - 逆境中的应变与决策**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 立即评估现状，主动制定多套备选方案并推动共识，聚焦解决问题 |
| 4-7分 | 会尝试分析问题、寻求帮助或调整部分计划 |
| 0-3分 | 表现出放弃、抱怨或完全等待上级指令 |

**Q9 - 持续高压下的动力维持**

| 分数段 | 评价标准 |
|--------|----------|
| 8-10分 | 展示成熟心态管理（如重定意义、建立支持系统）与灵活的资源策略，保持核心产出 |
| 4-7分 | 能通过分解任务、寻求短期激励来维持运作 |
| 0-3分 | 仅表示"坚持"或"硬扛"，缺乏具体方法 |

### 7.4 关键词评分规则

**正向关键词（每个+2分）**：
调整、改进、学习、克服、坚持、复盘、成长、总结、应对、突破

**负向关键词（每个-2分）**：

## 九、评分计算规则

### 9.1 单题评分计算

```
单题评分 = 基础分(5分) + 长度调整 + 关键词评分 + 相关性评分 + 结构化评分
```

**计算步骤**：

| 步骤 | 规则 | 分值范围 |
|------|------|----------|
| 1. 基础分 | 每题起步分 | 5分 |
| 2. 长度调整 | <10字:-3, <30字:-1, >100字:+1, >200字:+2 | -3 ~ +2 |
| 3. 关键词评分 | 正向关键词+2, 负向关键词-2, 总计限制 | -3 ~ +3 |
| 4. 相关性评分 | 回答与问题的相关程度 | 0 ~ +2 |
| 5. 结构化评分 | 有编号/逻辑词/总结 | 0 ~ +1 |
| **最终分数** | 四舍五入，限制在0-10 | **0 ~ 10** |

### 9.2 维度得分计算

```
维度得分 = (三题平均分 / 10) × 维度权重
```

**计算示例**：

| 维度 | 三题得分 | 平均分 | 权重 | 维度得分 |
|------|----------|--------|------|----------|
| IQ | 8, 9, 7 | 8.0 | 50 | (8.0/10)×50 = 40分 |
| EQ | 7, 8, 6 | 7.0 | 20 | (7.0/10)×20 = 14分 |
| AQ | 8, 7, 8 | 7.67 | 15 | (7.67/10)×15 = 11.5分 |
| MQ | 9, 8, 9 | 8.67 | 15 | (8.67/10)×15 = 13分 |

### 9.3 总分计算

```
总分 = IQ得分 + EQ得分 + AQ得分 + MQ得分
满分 = 50 + 20 + 15 + 15 = 100分
```

### 9.4 评级标准

| 总分范围 | 评级 | 说明 |
|----------|------|------|
| 90-100分 | 优秀 | 四维度表现均衡优秀 |
| 80-89分 | 良好 | 整体表现良好，个别维度有提升空间 |
| 70-79分 | 中等 | 整体表现一般，部分维度需加强 |
| 60-69分 | 及格 | 存在明显短板，需要重点关注 |
| 60分以下 | 待提升 | 多维度表现不足，建议慎重录用 |
